# Hamming Code 

Консольная программа для кодирования и декодирования данных с использованием кода Хэмминга.

Позволяет обнаруживать и исправлять ошибки, возникающие при передаче или хранении информации.

## Возможности
- кодирование данных с добавлением контрольных бит  
- декодирование с исправлением ошибок  
- автоматическое исправление одиночных ошибок  
- работа с произвольным размером блока  

## Использование

### Кодирование
1. Подготовьте файл `hemming.txt`  
2. Запустите программу и выберите `1`  
3. Введите размер блока  
4. Получите:
   - `hem_cod.txt` — закодированный файл  

### Декодирование
1. Используйте `hem_cod.txt`  
2. Запустите программу и выберите `2`  
3. Введите тот же размер блока  
4. Получите:
   - `hem_decod.txt` — восстановленный файл  

## Принцип работы
Данные разбиваются на блоки, в которые добавляются контрольные биты (позиции 2ⁿ).  
При декодировании вычисляется синдром ошибки, позволяющий определить и исправить повреждённый бит.

## Особенности
- корректно исправляются только одиночные ошибки  
- множественные ошибки не гарантированно исправляются  
- увеличивается размер данных за счёт контрольных бит  

## Технологии
C (работа с битами и файлами)
